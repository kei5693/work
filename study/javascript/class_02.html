변수

  • 수치나 문자들을 저장해 주는 '상자'
  • 대소문자를 구별
  • 숫자로 시작 할 수 없다.
  • 영문자, 언더바 (_), 달러기호(&)로 시작해야 한다.

변수의 형(type, 데이터의 종류)

  • 변수에 대입하는 값에 따라 변수형이 자동으로 정해진다.
  • typeof 연산자를 사용하면 변수의 형을 확인 할 수 있다.

데이터형

문자열형

  • 하나 이상의 문자를 나열한 것
  • "" 큰 따옴표, '' 작은 따옴표로 둘러싼다.
  • typeof 연산자를 실행하면 string 반환

문자열의 연결

+ 연산자를 사용

큰따옴표, 작은 따옴표 사용

문자열을 큰 따옴표로 감쌌을때 그 안에는 같은 큰 따옴표가 올 수 없다. 작은 따옴표도 동일히다.

이스케이프 시퀀스

문자 앞에 이스케이프 문자(\)붙여서 이스케이프 시퀀스라는 특수 문자를 사용하면 따옴표 안에서 같은 따옴표를 사용 할 수 있다.

데이터 형 변환

다른형 연결하기

문자열 + 숫자형 = 숫자형이 문자형으로 자동 형변환

숫자형 + 논리형 = 논리형이 숫자형으로 자동 형변환, true는 1, false는 0으로 취급

형변환

자바스크립트 내장함수를 사용

parseInt() : 문자열을 정수로 변환.(소수점 절삭)

parseFloat() : 문자열을 소수로 변환.

toString() : 문자열로 변환.

산술 연산자

대입 연산자

변수에 값을 대입한다.

증가, 감소 연산자

변수를 1씩 증가, 감소 시킬때 사용

위치에 따른 차이

변수 앞에 사용하면 전치, 뒤에 하용하면 후치라고 불리며, 연산을 수행하는 타이밍이 달라진다.

비교 연산자

조건을 만족하는 경우 참(true), 아니면 거짓(false)

삼항연산자

조건식의 결과에 따라 다른 처리

논리연산자

조건을 붙여 실행

연산자 우선 순위
